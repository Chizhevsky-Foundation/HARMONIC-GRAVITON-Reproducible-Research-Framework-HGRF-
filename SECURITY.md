# SECURITY.md — Política de seguridad

Responsable del proyecto: Fundación Chizhevsky  
Contacto para reportes de seguridad (confidencial): security@chizhevsky-foundation.org  
Alternativa: ia.mechmind@gmail.com

Resumen
-------
Reporta vulnerabilidades o exposiciones de datos sensibles de forma privada al correo arriba. No publiques detalles sensibles en issues públicas.

Proceso de reporte
1. Envía un correo a security@... con: resumen, impacto, pasos para reproducir (si aplican) y evidencia adjunta (capturas, logs, commits).
2. Recibirás acuse de recibo en 72 horas.
3. Investigación interna y remediación con cronograma (30 días hábiles o menor si es crítico).
4. Coordinación para divulgación responsable (si aplica).

Divulgación responsable
- No publiques explotables antes de que se aplique el parche o mitigación.
- En casos de fuga de datos personales, se seguirán obligaciones legales vigentes (notificación de brechas, etc.).

Firmado: Comité de Seguridad — Fundación Chizhevsky
